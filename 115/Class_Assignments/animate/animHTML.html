This is a simplistic webpage set up to demonstrate some animation examples. Moving objects can be useful to grab the audience's attention in a websites, especially when used tastefully.

The first example is an outlandish display of background colors for the body. I wanted to make the changes very evident. Remember that in actual websites you would want to be subtle.

The second animation also creates a greater spectacle than you would use on a website. A more subtle variation might be to move within a smaller amount of space.

The third example is pretty cool, in my opinion. It spins and transforms a picture when hovered over.

In the fourth example, the picture is reduced when the mouse hovers over it. The transition is smooth.
